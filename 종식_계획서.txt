[JavaScript]
__예매 페이지
1. 캘린더 아이콘 클릭 시 현재 날짜 표시(API이용)
2. 세부 극장 선택 시(e.g. 서울 - 강남) 현재 위치에서 강남극장까지 길찾기 페이지 이동 후 빠른길 찾기(API이용)
3. 상영 가능 시간 범위에 따라 시간 nav bar 범위 selected 폰트로 바뀜
4. 좌석선택 버튼 클릭 시 현재 선택한 영화 알려줌 (상세정보, 관람후기, 평첨, 공연장정보 // modal)
5. 영화 최대 2개 선택 가능 / 극장 최대 2개 선택 가능 / 시간은 모든 영화와 극장 render

__좌석 선택 페이지

__영화 페이지
1. 영화 포스터 클릭 시 예매 사이트 이동
2. 좋아요 클릭 시 JSON or localStorage에 저장 후 레이아웃 업데이트
3. 예매 클릭 시 예매 사이트 이동
4. 버튼 누를 시 좋아요 순으로 정렬 / 다시 누를 시 초기화

[Flow]
{예매 페이지 // 영화, 장소, 시간 선택} -> {좌석 선택 페이지 // 인원수, 좌석, 제휴할인 선택}
	-> {로그인 페이지 // 정상 or 회원가입 or } -> {예매 확인 페이지}

{영화 페이지}   -> 	{빠른예매 // 예매 페이지 이동}
	       ->	{현재개봉작 // 개봉작만 보여줌}
	       ->	{즐겨찾기 // JSON or localStorage에 저장 후 나중에 열람 가능}
